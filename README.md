# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **Sales Dashboard** built using Power BI to analyze and visualize business performance across different regions, categories, and time periods.
The dashboard helps in understanding key sales trends, identifying top-performing products, and supporting data-driven decision-making.

## 🎯 Objectives

  Analyze overall sales and profit performance
  Identify top-selling products and categories
  Track regional sales distribution
  Understand customer purchasing patterns


## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization & dashboard creation
* **Microsoft Excel** – Data cleaning & preprocessing

---

## 📂 Dataset

* Superstore Sales Dataset (Excel format)
* Contains information about:

  * Orders
  * Customers
  * Products
  * Sales & Profit

---

## 📈 Key Insights

* 📌 Identified top-performing product categories contributing to revenue
* 📌 Analyzed region-wise sales distribution
* 📌 Detected sales trends over time
* 📌 Highlighted profit vs loss segments

---

## 📊 Dashboard Features

* Interactive filters and slicers
* Region-wise and category-wise analysis
* Time-based sales trends
* KPI indicators for quick insights


## 📁 Project Files

* `SuperStore_sales.pbit` → Power BI dashboard file
* `SuperStore Sales DataSet.xlsx` → Raw dataset
* `SuperStore_sales.pdf` → Dashboard report

## 🚀 How to Use

1. Download the `.pbit` file
2. Open it in Power BI Desktop
